CSS [ Attribute ] Selector

Selector Description
[attribute] The selector is used to select elements with a specified attribute.
[attribute="value"] The selector is used to select elements with a specified attribute and value.
[attribute~="value"] The selector is used to select elements with an attribute value containing a specified word.
[attribute|="value"] The selector is used to select elements with the specified attribute, whose value can be exactly the specified value, or the specified value followed by a hyphen (-).
[attribute^="value"] The selector is used to select elements with the specified attribute, whose value starts with the specified value.
[attribute$="value"] The selector is used to select elements whose attribute value ends with a specified value.
[attribute*="value"] The selector is used to select elements whose attribute value contains a specified value.
